## 🎯 Purpose

> “If they’re going to waste our time, we’ll waste theirs — with style.”

## 📦 Installation

Set up your virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
pip install signalwire-agents
pip install python-dotenv
```

## Run the Agent
```bash
python3 signalwire.py
```
